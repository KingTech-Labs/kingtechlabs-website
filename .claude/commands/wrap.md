# Wrap Session

Do the following steps in order:

## 1. Commit and push any unpushed changes

Check git status. If there are uncommitted changes, stage and commit them with a short summary message describing what was changed. Then push to GitHub. If everything is already committed and pushed, skip this step.

## 2. Write a session summary to CLAUDE.md

Replace the `## Last Session` section at the top of the project's `CLAUDE.md` file with a fresh summary containing:
- What we were working on
- What was completed this session
- What is pending or next up
- Any important decisions made
- The last commit hash

This section should give enough context to resume the session cold without re-reading the conversation.
